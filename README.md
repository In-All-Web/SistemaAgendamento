# Sistema de Agendamento

Este projeto visa ajudar todos os profissionais a gerir melhor as marcações e eventos que têm no seu negócio.

## O sistema:
1- Agenda online
2- Página de marcações
3- Gestão da empresa
4- Gestão de equipas
5- Gestão de clientes
6- Lembrete de SMS
7- Estatísticas
8- Marcações 24/7
9- Marketing integrado
10- Pagamentos MBWay.
11- Ponto de venda
12- Google review
