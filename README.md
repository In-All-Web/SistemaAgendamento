# Sistema de Agendamento

Este projeto visa ajudar todos os profissionais a gerir melhor as marcações e eventos que têm no seu negócio.

## O sistema:
- Agenda online
- Página de marcações
- Gestão da empresa
- Gestão de estabelecimentos
- Gestão de equipas
- Gestão de clientes
- Lembrete de SMS
- Estatísticas
- Marcações 24/7
- Marketing integrado
- Pagamentos MBWay.
- Ponto de venda
- Google review
